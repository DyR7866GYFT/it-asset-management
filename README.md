# Gestión de Activos de TI 💻

Una aplicación web SPA desarrollada en React y TypeScript diseñada para optimizar el control, registro y seguimiento de la infraestructura tecnológica en entornos de soporte técnico local o entornos comerciales.

El objetivo principal de este proyecto es demostrar habilidades en la estructuración de soluciones para problemas reales de administración TI, gestionando estados, componentes reutilizables y persistencia de datos del lado del cliente.

## 🚀 Características Clave

* **Registro Completo de Activos:** Formulario dinámico para añadir equipos con datos críticos de soporte (Nombre del equipo, Tipo, Dirección IP, Estado operativo).
* **Filtros en Tiempo Real:** Búsqueda y segmentación rápida de dispositivos según su estado (Activo, En Mantenimiento, De Baja) para agilizar la toma de decisiones.
* **Persistencia de Datos Local:** Implementación de `LocalStorage` para simular una base de datos local, garantizando que la información se mantenga al recargar la aplicación.
* **Arquitectura Escalable:** Estructura limpia basada en componentes modulares que facilitan el mantenimiento del código.

## 🛠️ Tecnologías Utilizadas

* **Frontend:** React.js
* **Tipado Dinámico:** TypeScript
* **Herramienta de Construcción:** Vite
* **Estilos:** CSS Modules / CSS Puro
* **Almacenamiento:** Web Storage API (LocalStorage)
